# Movie Explorer

A web application built using HTML, CSS, and JavaScript that allows users to search for movies, series, and games using the OMDb API.

## Features

* Search movies by title
* View movie posters, titles, and release years
* Filter results by category:

  * Movies
  * Series
  * Games
* Pagination support using Previous and Next buttons
* Display total number of search results
* Responsive card-based layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* OMDb API

## Concepts Practiced

This project was built to practice core JavaScript concepts:

* Async/Await

* Event Listeners
* DOM Manipulation
* Array map()
* Destructuring
* Pagination Logic
* Dynamic Rendering

## Project Structure

movie-explorer/

* index.html
* style.css
* main.js

## How to Run

1. Clone the repository.
2. Get an API key from OMDb API.
3. Add your API key in `main.js`.
4. Open `index.html` in your browser.
